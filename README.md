# 📱 Used Phone Resale Price Predictor

An Artificial Neural Network (ANN) built with Keras/TensorFlow that predicts the resale price of a used phone based on its specs, age, condition, and market demand — served through an interactive Streamlit web app.

## 🎯 Project Overview

- **Problem type:** Regression (predicting a continuous price value)
- **Model:** Feed-forward ANN (Dense → BatchNorm → Dropout, 3 hidden layers)
- **Dataset:** 1,000,000 rows of used phone listings (brand, specs, age, condition, damage history, market demand, etc.)
- **Final performance:** R² ≈ 0.98, MAE ≈ ₹1,600–1,700

## 🧠 How It Works

1. **Data cleaning:** dropped columns with negligible predictive value (`screen_size_inches`, `usage_hours_per_day`, `city_tier`, `seller_type`, `charger_available`, `release_year`, `processor_score`, `camera_score`) based on correlation and feature-importance analysis. Kept `original_price`, `age_months`, `market_demand_score`, `battery_health`, and `condition` — the strongest predictors.
2. **Encoding:** categorical columns (`brand`, `model`, `condition`, `os_type`) one-hot encoded.
3. **Scaling:** both features and target scaled with `StandardScaler` for stable training.
4. **Model:** a 3-hidden-layer ANN (128 → 64 → 32 → 1) with BatchNormalization and Dropout for regularization.
5. **Training:** Adam optimizer, MSE loss, with `EarlyStopping` and `ReduceLROnPlateau` callbacks.
6. **Deployment:** the trained model + scalers are loaded into a Streamlit app that takes phone details through a form and returns a live price prediction.

## 🚀 Running Locally

1. Clone this repo:
   ```bash
   git clone https://github.com/<your-username>/phone-price-predictor.git
   cd phone-price-predictor
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. **(Optional) Retrain the model** — only needed if you want to retrain from scratch. Download the dataset and place `used_phone_price_prediction_1M.csv` in this folder, then run:
   ```bash
   python train_model.py
   ```
   This regenerates the files in `models/`. If you just want to run the demo, skip this — the trained model is already included in `models/`.

4. Run the app:
   ```bash
   streamlit run app.py
   ```
   Open the link shown in your terminal (usually `http://localhost:8501`).
