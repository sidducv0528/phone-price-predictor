## About the Deployment

The deployed web application allows users to estimate the resale price of a smartphone by entering its specifications and condition.

The application performs the following steps:

1. Accepts smartphone specifications from the user.
2. Preprocesses the input data.
3. Loads the trained Artificial Neural Network (ANN) model.
4. Predicts the estimated resale price.
5. Displays the predicted price in an interactive Streamlit interface.

## How to Run Locally

### Clone the Repository

```bash

cd Phone-Price-Prediction-Using-ANN
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Launch the Application

```bash
streamlit run app/app.py
```
